
The Week 12's lab implementing signup and login screens.

The feature added an extra one screen for users to enter user name and
password, the method is set the application content as the new screen,
if user enter the correct username and passwords, they will allow to get
into the normal week11 world cup application to let them to do some method.
Also, they are allow to log out from the world cup and reenter.

The application has a default user name and password "admin".
However, all users can enter the new name and password in the same place
and click sign up to store their info to the application's list. And the
username have to more than 1 and same username is not allowed in the app.
